# aitools

A collection of useful AI-related tools for programmers.

## What's in here

- **Custom Skills** -- Claude Code skills for common workflows
- **Knowledge Base** -- Curated references and prompts
- **Status Line Scripts** -- Bash scripts for Claude Code status line
- **Misc Utilities** -- Other handy AI dev tools

## Usage

Browse the directories and grab what you need. Each tool includes its own usage instructions.

## Contributing

Drop your tools in the appropriate directory and open a PR.
